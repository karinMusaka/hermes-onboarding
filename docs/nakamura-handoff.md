# 中村さん向け案内文（コピペ用）

## Slack DM で送る文面

```
中村さん

Hermes（AI アシスタント）のセットアップ準備ができました！
ターミナルに以下の1行を貼り付けて Enter を押すだけです。

あとは画面の案内に沿って進めれば OK です。
途中で API キーを聞かれたら、以下を貼り付けてください:

API キー: [ここに発行したキーを貼る]

何かあったら聞いてください 👍
```

## 口頭で説明するとき

1. 「Cmd + スペース で Spotlight 開いて、"ターミナル" って打って」
2. 「黒い画面が出たら、このコマンドをコピーして貼り付けて」
3. 「Enter 押したらあとは自動。パスワード聞かれたら Mac のパスワード」
4. 「API キーって聞かれたら、Slack で送ったやつを貼って」
5. 「終わったら hermes って打てば AI と会話できるよ」
